This project focuses on comparing the KNN and Logistic Regression models when it comes to predicting the number of occupants in a room.
The dataset was taken from the UCI ML repository: https://archive.ics.uci.edu/dataset/864/room+occupancy+estimation
-
Each file contains specific elements of the overall project:
* Logistic_Regression_Model.ipynb
* * logistic regression model
* KNN.ipynb
* * Data Exploration Visualzations (bar graph, heatmap, pairplots)
  * KNN model
